Experiencias de la carrera

Mis experiencias

En mi primer dia como estudiante de informatica siempre estaba nervioso de no lograr mantener el ritmo de las clases, pero gracias a los buenos maestros que se me presentaron en la escuela logre aprender muchas cosas las cuales nunca imaginaba aprender en una "preparatoria", logre aprender programacion en un nivel decente y en distintos lenguajes, consegui amigos exelentes e inolvidables, experiencias enriquesedoras para mi persona, entre otros, en definitiva, no me arrepiento de aprovechar la oportunidad de estudiar en una buena escuela como lo fue esta.

¿Recomiendas estudiar en el politecnico?

definitivamente, si, es una "preparatoria" que no solo te da las materias basicas de una, tambien aumenta tus conocimientos en las carreras que tu eliges, muchas otras escuelas tienen este tipo de educacion en la cual se escogen talleres pero en realidad es muy poco lo que enseñan, en cambio, el politecnico se enfoca totalmente a enseñarte lo que tu carrera es y todo lo que necesitas para lograr su meta "crear algo mas que un obrero pero menos que un ingeniero".

Materias dentro de la carrera

Como alumno casi graduado del politecnico quisiera dejar algun listado de materias las cuales en mi consideracion son las mejores y las mas utiles

CCNA

CCNA 1
CCNA es la materia la cual te introduce al mantenimiento de redes informaticas en empresas y/o particulares, esta materia comienza en 5to grado
CCNA 2
Al igual que el primer CCNA el segundo se lleva dentro de las clases "normales" aumentando un poco mas la informacion y dificultad de lo que se pide realizar
CCNA 3
CCNA 3 a diferencia de los otros dos ya mencionados es una materia "extra" la cual puede ser habilitada por un maestro
CCNA 4
Al igual que CCNA 3 el 4 tambien es una materia "extra".

Lenguajes de programacion

C
C es el lenguaje de programacion el cual te prepara e introduce a este tipo de actividades, es bastante sencillo de utilizar pero no contiene muchas utilidades
JAVA
JAVA a diferencia de C es mas complicado pero a cambio, tiene muchas utilidades y existen infinitas posibilidades de realizar algo si sabes como y que quieres hacer
ALICE
ALICE se enfoca mas a la animacion usando partes de codigo en java

Paginas web

HTML
HTML es el lenguaje utilizado para realizar todo ripo de paginas web, una manera de ver esto o un ejemplo es en esta misma pagina dando clic derecho > ver codigo fuente, todo el codigo que aparece a continuacion sera un conjunto de html, CSS y PHP
PHP
PHP se utiliza para agregar mas funciones a las paginas web, como lo son registros en base de datos, por ejemplo, inicio de sencion en facebook
CSS
CSS se puede explicar de una manera sencialla al decir que es el decorar y acomodar la pagina web, fondos, colores, tipo de letra, etc, todo la parte visual es CSS
Base de datos
Las bases de datos aun si no son exclusivas para paginas web son bastantemente usadas en conjunto con php para los registros o funciones dentro de estas
